Debris Removal Feature

UI Wireframes & Design Mockups

Overview: This feature adds debris removal vehicles (DRVs) to the Kessler Simulation game, allowing players to actively clean up orbital debris. DRVs come in two types: Cooperative (targets large, trackable debris) and Uncooperative (targets small, untracked debris). The following wireframes demonstrate the UI updates needed to support this strategic gameplay mechanic.
